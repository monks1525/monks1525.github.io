我的第一篇博客文章

第一章 我创建了我的博客

这是根据hexo来创建的，基于node.js

第二章 博客创建流程

第一步

下载nodejs，windows最新版本，网址：nodejs.cn，直接安装，我安装在我的D盘，安装完成后有两个组件，一个是node.js，另一个npm包管理器（这个是搭建hexo博客必须的东西）。

第二步

因为我观看的视频是macOS（x64）系统的，但我所用的电脑是windows的，所以接下来写下的东西是windows系统的。

打开cmd，输入node -v查看版本，输入npm -v查看版本，因为hexo是基于node.js来生成的，所以这是前置部分。现在就可以真正地安装hexo的博客框架了。

第三步

安装之前，我们是借助npm包管理器来进行安装，但是由于国内的镜像源速度很慢，所以我们需要利用npm来安装一个cnpm（淘宝的那个源），这样比较快。

代码如下：

npm install -g cnpm –registry=http://registry.npm.taobao.org

笔记：-g意思是全局安装，registry意思是镜像源，=就是指向淘宝

第四步

然后敲击cnpm，就能出来各种信息，表示就有这个工具了，然后敲击以下代码：

cnpm install -g hexo-cli

这一步就是真正安装hexo博客了，需要记住hexo-cli

然后他会自己去下载

下载安装完成后，可以用hexo -v来验证一下

第五步

利用chdir来查看目录，mkdir来新建文件夹，比如我新建一个blog的文件夹，输入以下代码：

mkdir blog

文件夹生成在chdir的目录列表下，当然也可以利用cd + 路径来进入所要去的文件夹，然后建立blog

cd blog/

就是进入blog这个文件

第六步

初始化博客：hexo init

可以利用 dir（mac是 ls -l） 来查看生成了哪些东西（目录）

第七步

启动我们的博客：hexo s或server

生成：localhost:4000

在4000端口生成

第八步

输入hexo n ”我的第一篇博客文章“，就生成了一个名为”我的第一篇博客文章“的markdown文档，就可以开始编辑啦，目前我所写的所有东西，就是在这个markdown文档里写的，利用Typora工具。还有sublime，只要是一个markdo的编辑器就可以了。

退出只要输入：wq（在vim编辑器里）

第九步

现在在blog文件夹里，需要退两层，输入：

cd ../..

就点击返回两次

接下来的操作是边看视频边学的，所以会直接输入命令，并带上自己的笔记。

清理博客（为什么要清理？清理静态文本）：hexo clean

然后再生成一下（重新生成静态文本）：hexo genera或者简写g

然后启动hexo：hexo s

就可以看到我写的第一篇博客文章了
